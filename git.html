Git

Version Control

Git is a version control program popular in industry and the open source community. It helps teams to manage multiple people contributing to the same code base and has tools for reseolving conflicts in different versions. It utilizes branches which allows for multiple streams of development as well as maintaining a stable version. I have used Github, one of the largest Git hosting services, in many personal and some team projects. Most notably, I used it to work on my 10 person software engineering team which, over the course of writing our hospital directory application, wrote 21,120 lines of Java.
