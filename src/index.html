The Solar System

In the background, you can see a simulation of the Solar System. This is all running locally in your browser using two-body orbital mechanics that I've implimented in Javascript. As you scroll down, the distance scale is decreased and time accelerates allowing you to see more of the system.

If you're currious, you can see the astronautics code that runs this. Javascript isn't typically used for scientific computing, so if you want to look at a similar python program of mine its also shown below.
